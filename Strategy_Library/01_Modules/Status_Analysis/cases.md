# Case Study: 兵哥 (MVP User)

## 1. 用户背景
*   **用户**：兵哥，男，大厂工程师/军人背景，条件好但不懂女生。
*   **Crush**：校招生同事，回避型/压力敏感型。
*   **关系**：T1 备胎/供养者陷阱。A低C高。

## 2. 核心问题
*   **致命伤**：需求感爆棚。言语中透露“没你我不行”，强行送礼（零食/机械键盘）。
*   **对方反馈**：客气但冷淡，拒绝收礼，回消息慢。

## 3. 现状分析 (Status Analysis) 输出示例
> **军师小话**：兵哥，你的过度热情正在把“维持现状”变成“降级风险”。
>
> **📍 战局坐标**：`T1 - 供养者/备胎陷阱`
> *   **A低**：她极少主动，回复factual。
> *   **C高**：回表情包，开玩笑要手机玩（把你当无害好人）。
>
> **🚑 症结诊断**
> *   🔴 **致命伤**：过度暴露需求感（证据："没有手机我也没啥可玩的" -> 压力源）。
> *   🟡 **隐患点**：自我感动式付出（强送零食）。
>
> **🎯 破局目标**：消除压力，去“舔狗”化。让对方觉得你是有独立生活的高价值战友。
>
> **🗺️ 作战规划**：`阶段一·冷冻降压期`
> *   **目的**：止损，重建安全感。
> *   **里程碑**：她主动开启非工作话题。
> *   **原则**：停止主动，字数砍半。

## 4. 行动指南 (Action Guide) 输出示例
> **当前战役**：❄️ 阶段一·冷冻降压
>
> **📋 任务卡片：镜像撤退 (Operation Mirror)**
>
> **简报**
> *   目标：通过**行为后撤**制造反差，引发好奇。
> *   时机：即刻开始，持续 48-72h。
>
> **执行 SOP**
> 1.  **绝对禁言**：不主动发，不赞不评。
> 2.  **被动防御**：
>     *   聊正事 -> 公事公办。
>     *   聊闲天 -> 延迟 30-60 分钟，字数砍半。
> 3.  **朋友圈展示**：断联第二天晚，发积木图，配文“专注时时间过得最快”（暗示独处充实）。
>
> **预案**
> *   🔴 没找你 -> 好事！说明 T1 判断正确，继续冷冻。
> *   🟢 问“消失了？” -> 回复“最近训练晕了，刚活过来”（轻松抛回）。
>
> **锦囊**
> *   现在的你是打折零食。要挂出“暂停营业”，她才会来看发生了什么。
